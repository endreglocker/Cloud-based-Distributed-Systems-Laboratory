# Cloud-based-Distributed-Systems-Laboratory

The goal of this project to create a web-based image-viewer for multiple users.

## Requirements

1. Users
    - [X] Create user
    - [X] Login
    - [X] Exit
    - [X] Upload / Delete images only allowed for loged in users
2. Images
    - [X] Name: max 40 characters
    - [X] Upload date: YY-MM-DD HH:MM
    - [X] Upload / Delete new image
3. Image-View
    - [X] List image names and dates
        - [X] Order by names
        - [X] Order by dates
    - [X] Cicking on item: reveals image
